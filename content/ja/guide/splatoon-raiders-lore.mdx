『スプラトゥーン・レイダース（Splatoon Raiders）』の発表は、インクリングやオクトリングのコミュニティに衝撃を与えました。本作は、シリーズの歴史をかつてないほど深く掘り下げる、スタンドアロンのシングルプレイヤー体験を約束しています。専用のスピンオフ作品として、従来のナワバリバトル中心のスタイルから離れ、世界観の構築やキャラクター主導の物語に重点を置いています。ポスト・アポカリプス（終末後）の地球の歴史や、進化した頭足類たちの歩みを理解する上で、**『スプラトゥーン・レイダース』の伝承**を知ることはファンにとって不可欠です。

この新たな旅の舞台は、古代の秘密と環境ストーリーテリングに満ちた辺境の群島「スパイラライト諸島」です。虚空に浮かぶプラットフォームを利用していた過去のキャンペーンとは異なり、本作では地に足のついた、リアリティのある世界に焦点を当てています。**『スプラトゥーン・レイダース』の伝承**を読み解くことで、ファンに人気の3人組「すりみ連合」の動機や、「メカニック」としてのみ知られる新主人公の出自を明らかにすることができます。このガイドでは、2026年時点での設定、キャラクター、そしてシリーズの物語のトーンの変化について、現在判明しているすべてを解説します。

## スパイラライト諸島：新たなる辺境

『スプラトゥーン・レイダース』の主な舞台はスパイラライト諸島です。初期の物語の詳細によると、これらの島々は単なるランダムな陸地ではなく、豊かな生態学的歴史を示唆する独特のバイオームを有しています。より「自然的」な環境への移行により、シリーズではこれまで見られなかったレベルの細部描写が可能になりました。プレイヤーは、一般的な箱の中から「ミステリーファイル」を見つける代わりに、放棄された作業場、キャンプ場、古代の遺跡といった、文脈のある場所でそれらを発見することになります。

| バイオームの種類 | 主な特徴 | 伝承における潜在的な重要性 |
| :--- | :--- | :--- |
| **熱帯の砂浜** | ヤシのような植物、透き通った水、沿岸の遺跡。 | 初期のインクリング定住地の残骸が含まれている可能性がある。 |
| **火山岩地帯** | 活発な溶岩流、黒曜石の形成、耐熱構造物。 | 諸島の産業史と密接に関わっている可能性が高い。 |
| **氷河地帯** | 永久凍土、凍りついた難破船、結晶の洞窟。 | 人類時代から保存された生物学的秘密が隠されている可能性がある。 |
| **古代の神殿** | 石造建築、光るグリフ、機械的なパズル。 | 先代文明や宗教的セクトの存在を示唆している。 |

## すりみ連合の新たな野望

フウカ、ウツホ、マンタローの3人は、**『スプラトゥーン・レイダース』の伝承**における中心人物として再登場しますが、その役割はバンカラ街のニュースキャスターからトレジャーハンターへと変化しています。『スプラトゥーン3』の「哺乳類の帰還」キャンペーンでは脇役に回ることが多かったすりみ連合ですが、『レイダース』では彼らの歴史とグループの絆が物語の推進力となります。

3人は特定の「お宝」を求めてスパイラライト諸島へとやってきましたが、その正体は謎に包まれています。この冒険は単なる富のためではなく、お互いの忠誠心を試すような個人的な使命であるようです。ゲームでは、彼らの家系や、現代の出来事よりずっと前に彼らの先祖がどのようにこの島々と関わっていたのかが探求されます。

## メカニック：謎に包まれた主人公

本作では初めて、プレイヤーは明確な職業的背景を持つキャラクター「メカニック」を操作します。キャラクターのカスタマイズは可能ですが、世界における役割は固定されています。メカニックの過去は「謎」とされており、一般的なインクリングを超える技術的知識を持っていることが示唆されています。

メカニックに関する**『スプラトゥーン・レイダース』の伝承**の中で最も重要な要素の一つは、その特殊な装備です。ギアの分析から、武器に対して「間に合わせ」のアプローチを取っていることが判明しており、これは彼らが長年にわたって「旧世界」の技術をスカベンジング（廃品回収）し、修理してきたことを示唆しています。

### メカニックの武器と道具

*   **手製のインクタンク：** 複数の武器キャニスターを保持できるようにカスタム構築されたタンクで、即座に武器を切り替えることが可能です。
*   **サルベージ・プロペラ：** 環境内に埋もれた資材や希少な遺物を探知するために使用されるデバイス。
*   **ロボットの相棒：** 戦闘や探索をサポートする、潜水艦のような姿をした歩行ロボット。このロボットにより、インクリングには不可能だった水中ゾーンの探索が可能になるのではないかと強く推測されています。

## 敵対者と超自然的な変化

コミュニティ内での大きな議論の的となっているのは、古代の神殿から放たれる超自然的な光によって召喚された巨大なサイクロンの存在です。これにより、シリーズが純粋なサイエンス・フィクション（AI、超進化、ホログラム）から、ファンタジーに近い要素へとシフトしているのではないかという議論が巻き起こっています。

**『スプラトゥーン・レイダース』の伝承**は、DJタコワサ将軍やクマサンのような従来のヴィランではなく、「自然の脅威」との対峙へと向かっているようです。しかし、ティザー映像の最後のフレームに映し出されたリアルで巨大なイカの目は、巨大な非知性的（あるいは超知性的）な生物学的存在が裏で糸を引いていることを示唆しています。

| エンティティ | 伝承における役割 | 説明 |
| :--- | :--- | :--- |
| **サイクロン** | 環境ハザード / プロットデバイス | 登場人物たちを島に閉じ込める巨大な嵐。 |
| **巨大なイカ** | 潜在的な敵対者 | ヒューマノイドであるインクリングとは異なる、リアルな頭足類。 |
| **新しい敵勢力** | 戦闘員 | オクトリアンから離れ、先住生物や機械的な敵が登場。 |

> **⚠️ 警告：** スパイラライト諸島は危険です。従来のオクトリアンの兵士がいないからといって安全なわけではありません。現地の「野生生物」や機械の守護者ははるかに攻撃的であり、ロボットの相棒を戦術的に活用する必要があります。

## 環境ストーリーテリングと没入感

開発者は、『オクト・エキスパンション』や『サイド・オーダー』からストーリーの伝え方を改善するためのインスピレーションを得ていることは明らかです。『スプラトゥーン・レイダース』では、キャラクターが静止画の上で話す「無線通信」形式が、フルアニメーションのカットシーンに大幅に置き換えられました。これにより、特にメカニックにおいて、単なるアバターではなく世界の一部として生きているような、より細やかなキャラクター表現が可能になっています。

この没入感は、プレイヤーの拠点となる「ラフト（筏）」にも及びます。諸島を進むにつれて、ラフトは新しい施設、NPC、回収された伝承の記録によってアップグレードされていきます。このハブワールドはあなたの発見に基づいて進化し、**『スプラトゥーン・レイダース』の伝承**がプレイヤーの行動に反応しているように感じさせてくれます。

スプラトゥーンの世界に関する公式アップデートについては、[スプラトゥーン公式サイト](https://splatoon.nintendo.com/)を訪れて、最新ニュースやパッチ情報を確認してください。

## 戦闘の進化：スプラシューターを超えて

『レイダース』はスタンドアロンのシングルプレイヤーゲームであるため、マルチプレイヤーのPvPバランス調整に縛られることがありません。これにより、**『スプラトゥーン・レイダース』の伝承**において、より強力で表現力豊かなアクションを正当化することが可能になりました。

1.  **強化版イカノボリ：** 高所の遺跡に到達するために使用される垂直飛行能力。
2.  **前方ダッシュ：** 敵のシールドを回避するために使用される素早い回避機動。
3.  **ショクワンダーの統合：** 移動と戦闘パズルの両方に使用される、恒久的なグラップリングフックツール。
4.  **クラフトシステム：** プレイヤーは「スクラップ」を集めて、武器のダメージ、インク効率、ロボットの能力を強化します。

## FAQ

**Q: 『スプラトゥーン・レイダース』は『スプラトゥーン3』の続編ですか？**

A: 同じタイムラインを舞台にしたスタンドアロンのスピンオフ作品です。前作のキャラクターが登場しますが、新しい場所と自己完結したストーリーに焦点を当てており、過去作の深い知識がなくても**『スプラトゥーン・レイダース』の伝承**を楽しめるようになっています。

**Q: 『スプラトゥーン・レイダース』のメインヴィランは誰ですか？**

A: 特定の個人名は明かされていませんが、ティザー映像では巨大でリアルなイカと謎の超自然的な嵐（サイクロン）が主な敵対勢力として描かれています。これまでの作品よりも「自然 vs テクノロジー」という対立軸が強まっているようです。

**Q: メカニックは水の中を泳げますか？**

A: インクリングは生物学的な構成上、依然として水に溶けてしまいます。しかし、メカニックのロボットの相棒は潜水艦のような外見をしており、これが水中探索セグメントの乗り物として機能すると多くのファンが予想しています。

**Q: オクトリアンはこのゲームに再登場しますか？**

A: 現在の情報では、シリーズは主要な敵勢力としてのオクトリアンから離れつつあるようです。『スプラトゥーン・レイダース』では、スパイラライト諸島の新しい先住生物や、前文明が残した古代の機械的な守護者に焦点が当てられています。
