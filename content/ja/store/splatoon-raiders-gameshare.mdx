## Switch 2におけるゲームシェアの仕組みを理解する

設定に入る前に、2026年時点での「いつも遊ぶ本体」システムの機能を理解することが重要です。すべてのニンテンドーアカウントは、1台のコンソールを「いつも遊ぶ本体」として指定できます。この本体では、そのアカウントが所有するゲームを、オフラインであってもどのユーザープロフィールでもプレイできます。それ以外の「2台目（サブ）」の本体では、ゲームを購入したアカウントのみがプレイ可能で、ライセンスを確認するためにアクティブなインターネット接続が必要になります。

**スプラトゥーン レイダースのゲームシェア**を成功させるには、あなたとパートナーで実質的に「いつも遊ぶ本体」を入れ替えることになります。これにより、プレイヤーAは自分のマシンでプレイヤーBのゲームをプレイでき、その逆も可能になります。

| 機能 | いつも遊ぶ本体 | それ以外の本体 |
| :--- | :--- | :--- |
| **オフラインプレイ** | 全ユーザーが利用可能 | **利用不可**（インターネット必須） |
| **ユーザーアクセス** | 本体の全プロフィール | **購入者のみ** |
| **認証** | 不要 | **必須**（オンラインチェック） |
| **スプラトゥーン レイダース** | ネイティブ4K/60fpsで動作 | ネイティブ4K/60fpsで動作 |

## ステップバイステップ：スプラトゥーン レイダースのゲームシェア設定方法

両方のプレイヤーが同時にゲームにアクセスできるように、以下の指示に従ってください。この方法は、「ソフトの中断」エラーを回避しながら**スプラトゥーン レイダースのゲームシェア**を行うための最も信頼できる方法です。

1.  **「いつも遊ぶ本体」の登録を解除する**: 自分のSwitch 2でニンテンドーeショップを開き、プロフィールアイコンを選択し、「いつも遊ぶ本体」までスクロールして登録を解除します。
2.  **2台目の本体でログインする**: 友人のSwitch 2で、あなたのニンテンドーアカウントにログインしてもらいます。
3.  **「いつも遊ぶ本体」に設定する**: 友人があなたのアカウントでeショップに入ると、その本体が自動的にあなたのアカウントの「いつも遊ぶ本体」になります。
4.  **ゲームをダウンロードする**: 友人はあなたの「再ダウンロード」リストからスプラトゥーン レイダースをダウンロードできるようになります。友人は**自分自身のプロフィール**を使ってプレイできます。
5.  **元の本体でプレイする**: あなたは自分のアカウントにログインすることで、元の本体で引き続きスプラトゥーン レイダースをプレイできます。この本体はあなたにとって「それ以外の本体」となるため、常にインターネットに接続している必要があります。

## スプラトゥーン レイダースのパフォーマンスへの期待

Nintendo Switch 2への移行により、グラフィックの忠実度が大幅に向上し、**スプラトゥーン レイダースのゲームシェア**体験にも影響を与えています。スプラトゥーン3の後期アップデートで見られたように、ハードウェアは現在、負荷の高いバンカラ街のハブや新しいレイダークラスのマップを含むすべてのモードで、固定60フレーム/秒（FPS）をサポートしています。

2026年のハードウェアの大きな利点の一つは、「ウルトラハイデフィニション（超高精細）」出力です。ゲームシェアを行っている場合、両方のプレイヤーが新しいハードウェアでプレイしていれば、この強化された解像度の恩恵を受けることができます。ゲームは高度なレンダリング技術を活用しており、遠くのNPCや環境の詳細でさえネイティブ60fpsを維持しています。これは、前世代で見られた30fpsの制限からの大きな飛躍です。

> **💡 ヒント：** 「それ以外の本体」でプレイする場合、システムが定期的にライセンスチェックを行うため、インターネット接続が安定していること（できれば有線LANアダプターの使用）を確認してください。信号が途切れるとマッチ中にラグが発生する可能性があります。

## オンラインマルチプレイとレイダースモード

スプラトゥーン レイダースでは、チームが希少なインクベースのリソースを奪い合う、ハイステークスな「レイダー」マッチが導入されています。**スプラトゥーン レイダースのゲームシェア**方式を使用する場合、異なるニンテンドーアカウントを使用していれば、両方のプレイヤーが同じオンラインマッチに参加できます。

### オンラインプレイの必須要件：
*   **Nintendo Switch Online (NSO)**: 両方のプレイヤーが有効なNSOサブスクリプションを持っている必要があります。ゲームを共有している場合は、ファミリープランを検討すると、両方のアカウントを低コストでカバーできます。
*   **バージョンの同期**: 両方の本体が最新バージョン（2026年4月現在、バージョン1.0.4）に更新されている必要があります。
*   **固有のプロフィール**: 「いつも遊ぶ本体」側で遊ぶ人は、所有者のプロフィールではなく、自分自身の固有のプロフィールを使用してプレイする必要があります。これにより「ユーザーがすでにプレイ中です」という競合を避けることができます。

| モード | ゲームシェア対応？ | 備考 |
| :--- | :--- | :--- |
| **レイダー・ガチマッチ** | はい | 個別のNSOアカウントが必要 |
| **サーモンラン：ネクストジェン** | はい | 60fpsの協力プレイに最適 |
| **ヒーローモード** | はい | いつも遊ぶ本体ならオフラインプレイ可 |
| **ローカル通信** | **いいえ** | ゲームシェア中は通常、リンクされた2台間でのローカル通信は無効になります |

## よくある問題のトラブルシューティング

完璧な設定であっても、**スプラトゥーン レイダースのゲームシェア**接続の管理中に障害に遭遇することがあります。ほとんどの問題は、「それ以外の本体」のチェックやプロフィールの競合に起因します。

*   **「このソフトは遊べません」**: これは通常、「いつも遊ぶ本体」側で所有者のプロフィールを使ってゲームをプレイしている場合に発生します。「いつも遊ぶ本体」のユーザーが別のプロフィールを使用しているか確認してください。
*   **頻繁な切断**: 「それ以外の本体」はオンラインでライセンスを確認する必要があるため、NATタイプの問題やWi-Fiの干渉によってゲームが終了することがあります。ルーターの設定を確認し、NATタイプAまたはBを目指してください。
*   **DLCへのアクセス**: 拡張コンテンツ「ディープシー・レイダース」を購入した場合、ゲームシェアのルールはDLCにも適用されます。「いつも遊ぶ本体」のユーザーはどのプロフィールでもDLCにフルアクセスできますが、所有者は「それ以外の本体」で自分のプロフィールを使用している間のみアクセス可能です。

より詳細なテクニカルサポートについては、アカウント管理とデジタル著作権に関する最新情報が掲載されている[任天堂公式サイトのサポートポータル](https://www.nintendo.com)をご覧ください。

## FAQ

**Q: スプラトゥーン レイダースのゲームシェアを使って、友達と同時に遊ぶことはできますか？**

A: はい、可能です。「いつも遊ぶ本体」が別のプロフィールで使用されており、「それ以外の本体」が実際にゲームを所有しているアカウントで使用されている限り、両方で同時にオンラインプレイが可能です。

**Q: スプラトゥーン レイダースのゲームシェアをすると、ランクやセーブデータに影響しますか？**

A: いいえ。セーブデータは個々のニンテンドーアカウントに紐付けられ、（NSO加入により）クラウドに保存されます。ゲームシェアはソフトを起動するためのライセンスを共有するだけであり、その中での進行状況は共有されません。

**Q: Switch 1とSwitch 2の間でスプラトゥーン レイダースをゲームシェアできますか？**

A: ライセンスの共有は可能ですが、スプラトゥーン レイダースはNintendo Switch 2専用に最適化されています。ゲームが「次世代機専用」タイトルの場合、初代Switchハードウェアではダウンロードもプレイもできません。

**Q: 「それ以外の本体」でインターネット接続が切れたらどうなりますか？**

A: ゲームが一時停止するか、起動に失敗します。「それ以外の本体」は、所有者がプレイしていることを確認するために常に接続を必要とします。旅行中などにプレイする予定がある場合は、そちらを「いつも遊ぶ本体」に設定しておくようにしてください。
