## L'origine du nom de code « Spiral »

Les noms de code internes chez Nintendo sont rarement choisis au hasard. Souvent, ils reflètent une mécanique centrale, le nom d'un protagoniste ou le cadre principal du jeu. Dans le cas du **nom de code spiral pour splatoon raiders**, le titre fait directement référence aux **Îles Spiralite**, l'archipel principal où se déroule le jeu. Ce lieu représente un changement significatif par rapport à l'étalement urbain de Chromapolis ou Cité-Clabousse, offrant aux joueurs un environnement plus tropical et axé sur l'exploration.

Le développement de Splatoon Raiders aurait commencé environ deux ans avant les premières fuites de 2024. Ce calendrier suggère que Nintendo cherchait à étendre la licence Splatoon au-delà du genre de tir compétitif pour en faire quelque chose de plus aventureux. La désignation « Spiral » a permis de garder le projet secret pendant que les développeurs se concentraient sur la verticalité unique et les mécaniques de « raid » qui définissent l'expérience.

## Splatoon Raiders : Gameplay et Univers

Splatoon Raiders n'est pas une guerre de territoire typique. En utilisant le **nom de code spiral pour splatoon raiders** comme fondation, les développeurs ont construit un jeu centré sur la chasse au trésor, les énigmes environnementales et le combat PvE (Joueur contre Environnement). Les Îles Spiralite sont conçues avec une topographie en « spirale », où les joueurs doivent gravir des sentiers de montagne sinueux et descendre dans de profondes tranchées sous-marines pour récupérer d'anciens artefacts Inklings.

La boucle de gameplay implique des « Raids », qui sont des missions instanciées pouvant être accomplies en solo ou avec une escouade allant jusqu'à quatre joueurs. Contrairement aux matchs rapides de 3 minutes de la série principale, ces raids peuvent durer entre 15 et 45 minutes, nécessitant une gestion minutieuse des ressources et une utilisation stratégique des outils de déplacement à base d'encre.

> **💡 Astuce :** Lors de l'exploration des Îles Spiralite, portez toujours une arme secondaire de type « Grappin-Splat ». La verticalité du terrain « Spiral » rend la nage standard difficile dans certaines zones rocheuses.

## Futurs noms de code : Que reste-t-il ?

Le **nom de code spiral pour splatoon raiders** étant désormais pleinement identifié, les fans tournent leur attention vers les noms restants sur la liste. Deux titres en particulier ont suscité d'intenses débats parmi les analystes de l'industrie : **SP Red** et **Anna**.

### Projet SP Red
La désignation « Red » a historiquement été associée à la franchise Mario (par exemple, « Red Star » pour *Super Mario Odyssey*). « SP » signifie probablement « Special » ou « Super ». Les théories actuelles suggèrent qu'il pourrait s'agir d'une « Édition Spéciale » de *Super Mario World* ou d'une « Édition Switch 2 » haute performance de *Super Mario Bros. Wonder*, ajoutant potentiellement le support 120 FPS et des améliorations HDR.

### Projet Anna
« Anna » est largement considéré comme la suite de *Nintendo Switch Sports*. Le raisonnement derrière cela est assez astucieux : le nom de code du premier *Nintendo Switch Sports* était « Elsa ». Étant donné qu'Elsa et Anna sont les célèbres sœurs de *La Reine des Neiges* de Disney, il est fort probable que Nintendo conserve cette convention de nommage pour la suite.

## FAQ

**Q : Splatoon Raiders remplace-t-il Splatoon 4 ?**

R : Non, Splatoon Raiders est considéré comme un titre dérivé (spin-off). Bien qu'il utilise le **nom de code spiral pour splatoon raiders**, il se concentre sur l'exploration PvE et les raids plutôt que sur le multijoueur compétitif traditionnel en 4v4 que l'on trouve dans les épisodes principaux.

**Q : Où le nom de code « Spiral » est-il apparu pour la première fois ?**

R : Le nom de code est apparu pour la première fois dans une fuite de 2024 provenant d'une source nommée Midori. Il figurait aux côtés de plusieurs autres projets Nintendo qui ont depuis été confirmés comme des titres Switch 2.

**Q : Puis-je jouer à Splatoon Raiders sur la Nintendo Switch originale ?**

R : Bien que des rumeurs circulent sur une sortie cross-gen, le projet « Spiral » a été spécifiquement développé pour tirer parti du matériel de la Nintendo Switch 2, en particulier pour ses environnements plus complexes et ses interactions d'encre basées sur la physique.

**Q : À quoi fait réellement référence le « Spiral » dans le nom de code ?**

R : Il fait référence aux **Îles Spiralite**, le cadre du jeu. Les îles sont caractérisées par leur géographie en forme de spirale et leurs sentiers sinueux que les joueurs doivent parcourir pendant les raids.
